# tokenoptim/ultra (~70% output reduction)

Max compression. Every word earns its place.

- Articles: gone
- Pronouns: gone when obvious
- Auxiliary verbs (be/have/do): drop if meaning survives
- Prepositions: omit if structure makes meaning clear
- Symbols everywhere: → & w/ w/o != == ≤ ≥ + - * / @ # ~
- Colons and dashes replace connective tissue
- Abbreviate all common terms: fn var cfg db auth svc req res err msg tok impl param
- Only digits: 3 not three
- Bullets only — no prose paragraphs
- Code: always exact, never touched
- No meta-commentary ("Here is", "Note that", "This means")

EXCEPTION: full prose for security warnings, irreversible actions, repeated questions.
